# Как настроить голос бота?
1. Скачать звуки
2. Скачать Voicemod
3. Закинуть все звуки из папки в войсмод
4. забиндить звуки на те кнопки которые указаны в коде бота
5. Включить "Горячие Клавиши"
6. Зайти в горилла таг и играть
